# Угадай число

Простая игра "Угадай число" с графическим интерфейсом на Python.

## Описание

Компьютер загадывает случайное число от 1 до 100, а игрок должен угадать его за минимальное количество попыток. После каждой попытки игрок получает подсказку "больше" или "меньше".

## Требования

- Python 3.6+
- tkinter (обычно входит в стандартную поставку Python)
