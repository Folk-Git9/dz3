# Угадай число

Простая и увлекательная игра "Угадай число", разработанная на Python с графическим интерфейсом.

## Описание проекта

Компьютер загадывает случайное число в заданном диапазоне, а игрок должен угадать его за минимальное количество попыток. После каждой попытки игрок получает подсказку - больше или меньше загаданное число.

## Функциональные возможности

- ✅ Генерация случайного числа компьютером
- ✅ Ввод предположений игроком
- ✅ Интеллектуальные подсказки после каждой попытки
- ✅ Возможность начала новой игры без перезапуска приложения
- ✅ Простой и интуитивно понятный интерфейс
- ✅ Отслеживание количества попыток
- ✅ Валидация вводимых данных

## Технические требования

- Python 3.6 или выше
- Библиотека Tkinter (обычно входит в стандартную поставку Python)

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/your-username/guess-the-number.git
cd guess-the-number
```

2. Запустите игру:
```bash
python main.py
```

Или для запуска графической версии:
```bash
python gui.py
```

## Как играть

1. Запустите игру
2. Введите число в текстовое поле
3. Нажмите "Проверить" или клавишу Enter
4. Используйте подсказки ("Больше" или "Меньше") для следующих попыток
5. Когда угадаете число, игра сообщит о победе и количестве попыток
6. Нажмите "Новая игра" для начала нового раунда

## Технические детали

### Основные модули

- **`main.py`** - Содержит класс `GuessTheNumberGame` с основной игровой логикой
- **`gui.py`** - Реализует графический интерфейс с использованием Tkinter
- **`utils.py`** - Вспомогательные функции для валидации и обработки данных

### Ключевые функции

- Генерация случайных чисел в заданном диапазоне
- Валидация пользовательского ввода
- Предоставление подсказок на основе сравнения чисел
- Управление состоянием игры и подсчет попыток
